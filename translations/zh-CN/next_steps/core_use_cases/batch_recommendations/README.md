Amazon Personalize 批量推荐
---

使用非同步批处理工作流从不需要实时更新的大型数据集中获得推荐。例如，您可以创建一个批量推理作业，以便为电子邮件列表上的所有用户获得产品推荐，或者获得项目目录中的项到项相似度（SIMS）。如需获得批量推荐，您可以通过调用 CreateBatchInferenceJob API 创建一个批量推理作业。
